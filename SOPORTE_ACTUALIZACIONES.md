# Soporte de Actualizaciones (mensaje para clientes)

## 1) Mensaje corto (copiar/pegar por WhatsApp)
Hola. La actualizacion normalmente se instala sola al aceptar Reiniciar ahora.

Si en tu equipo no se instala en ese intento:
1. Cierra BarberOS.
2. Abre la pagina de descargas oficiales.
3. Descarga el instalador mas reciente.
4. Ejecutalo encima de tu version actual (no desinstales).
5. Abre BarberOS nuevamente.

Tus datos locales se conservan.

## 2) Mensaje tecnico breve para cliente que reporta error
Gracias por avisar. El auto-update intento instalar pero Windows bloqueo el proceso.
Usa la recuperacion manual desde la descarga oficial y luego abrimos la app para validar version.
Si vuelve a fallar, enviame el archivo update.log para diagnostico.

## 3) Script interno para soporte (operador)
1. Confirmar version actual del cliente en la app.
2. Guiar instalacion manual del ultimo release.
3. Confirmar version final despues de abrir la app.
4. Si falla otra vez, pedir update.log.
5. Registrar incidente con: fecha, version origen, version destino, error mostrado.
